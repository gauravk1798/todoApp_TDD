# TDD-Driven Flutter ToDo App

This project is a clean, minimal Flutter ToDo app built using **Test-Driven Development (TDD)**.

---

## Features

- Add, toggle, and remove ToDo items
- Clean architecture
- `Provider` for state management
- TDD with:
  - ✅ Unit tests (services)
  - ✅ Provider tests
  - ✅ Widget tests
